# Snake Game 🐍

A classic Snake Game built in Python. Control the snake, eat food to grow longer, increase your score, and avoid colliding with yourself.

## Features

* Smooth snake movement
* Score tracking
* Increasing game speed as the score grows
* Random food spawning
* Self-collision detection
* Game Over screen
* Restart functionality

## Controls

| Key   | Action                  |
| ----- | ----------------------- |
| ↑     | Move Up                 |
| ↓     | Move Down               |
| ←     | Move Left               |
| →     | Move Right              |
| Space | Restart after Game Over |

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/snake-game.git
cd snake-game
```

### Create a virtual environment (optional)

```bash
python -m venv .venv
source .venv/bin/activate
```

### Install dependencies

```bash
pip install pygame
```

## Running the Game

```bash
python main.py
```

## Gameplay

* The snake starts with a length of 3.
* Eating food increases the snake's length.
* Each food item increases your score.
* The snake moves faster as the score increases.
* The game ends when the snake collides with itself.
